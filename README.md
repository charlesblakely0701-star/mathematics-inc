# Mathematics, Inc.

A virtual directory for the mathematicians of Mathematics, Inc. — sign up, manage your profile, and discover collaborators across departments.

> Work in progress. A full README (live URL, setup, env vars, schema notes, product assumptions, future improvements) lands with the deploy PR.

## Stack (so far)

- Next.js 16 (App Router) + TypeScript + React 19
- Tailwind CSS v4

## Local dev

```bash
pnpm install
pnpm dev
```

Then open http://localhost:3000.
