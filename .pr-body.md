Adds the rest of the auth UX: a `/login` page and a reusable sign-out button.

### Files
- `src/app/actions/auth.ts` — extends with:
  - `loginAction` — Zod-validates, calls `signIn("credentials")` with `redirectTo=/directory`, surfaces `Invalid email or password` for `AuthError`s
  - `signOutAction` — calls `signOut` with `redirectTo=/login`
- `src/app/(auth)/login/{page,login-form}.tsx` — server shell + client form (mirror of register)
- `src/components/sign-out-button.tsx` — server component that wraps `signOutAction` in a `<form>` (works without JS)

- `pnpm build` green (`/login` in route table)
- `pnpm lint` clean
