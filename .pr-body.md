Adds the directory grid — the primary view of the app.

### Files
- `src/lib/users.ts` — `getDirectory()` / `getProfile()` with an explicit `PUBLIC_USER_SELECT` allow-list. `passwordHash` is **never** selected here; the only place it's read remains the Credentials `authorize` callback. Also a tiny `getInitials()` helper for the avatar fallback.
- `src/app/(app)/layout.tsx` — auth-gated app shell: header with brand, Directory + My profile links, signed-in email, sign-out. Calls `requireSession()` for defense-in-depth (proxy is best-effort per Next 16 docs).
- `src/app/(app)/directory/page.tsx` — server-rendered grid, `force-dynamic` (a stale directory would be confusing), with a friendly empty state.
- `src/components/directory/user-card.tsx` — initials avatar, name, title, department badge, bio line-clamp, up to 3 interest tags with a `+N more` overflow.
